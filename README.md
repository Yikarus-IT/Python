# Python Learning Path

This repo is for learning Python from zero, then using that knowledge to build a Python backend for a future React admin panel.

## How We Will Learn

Each lesson has three parts:

1. A small concept explanation.
2. A runnable Python example.
3. Exercises that make you change code and test it in the terminal.

The goal is not to memorize syntax. The goal is to understand how data moves through code.

## Minimum Topics Before The Admin Panel

1. Running Python files from the terminal
2. Variables and basic data types
3. Strings and string formatting
4. Numbers and arithmetic
5. Booleans and comparisons
6. Lists
7. Dictionaries
8. Conditionals
9. Loops
10. Functions
11. None and truthiness
12. Errors and exceptions
13. Modules and imports
14. Reading and writing files
15. Virtual environments
16. Installing packages
17. Basic testing
18. Type hints
19. Classes and objects
20. Working with dates
21. JSON
22. HTTP and APIs
23. FastAPI basics
24. Databases with SQLite
25. PostgreSQL concepts
26. React talking to a Python API

After this, we can design and build the admin panel.

## Running A Lesson

From this folder:

```powershell
python lessons/01_basics/example.py
```

To run the exercises:

```powershell
python lessons/01_basics/exercises.py
```
