# Lesson 1: Running Python, Variables, and Printing

In this lesson, we learn the smallest useful Python program.

## Concepts

### Python file

A Python file usually ends in `.py`.

Example:

```text
example.py
```

When we run it, Python reads the file from top to bottom.

### print

`print()` shows something in the terminal.

```python
print("Hello")
```

### Variable

A variable is a name that points to a value.

```python
app_name = "Admin Panel"
```

Here:

- `app_name` is the variable name.
- `"Admin Panel"` is the value.
- `=` means "store this value under this name".

### Basic data types

Python values have types.

```python
name = "Ana"          # string: text
age = 31             # integer: whole number
height = 1.72        # float: decimal number
is_active = True     # boolean: True or False
```

## Important Python Habit

Python is case-sensitive:

```python
name
Name
NAME
```

Those are three different names.
